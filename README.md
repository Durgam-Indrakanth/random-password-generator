# Random Password Generator

This is a simple and interactive Password Generator built using Python.
The program lets you choose the password length and select which character types to include. It also validates user input and ensures only `y/n` responses are accepted.


## Features

* User-defined password length
* Option to include:
    * Uppercase letters
    * Lowercase letters
    * Digits
    * Symbols
* Input validation for `y/n` choices
* Generates a strong random password
* Easy-to-understand code structure


## How It Works

1. User enters desired password length
2. User selects character types to include
3. Program validates all inputs
4. A random password is generated from the selected character sets
5. Password is displayed
